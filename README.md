# Кабарман — Telegram Бот 📣

Справочник услуг и бизнеса Иссык-Кульской области.
Районы: Каракол · Ак-Суу · Тюп · Жети-Огуз · Тон · Чолпон-Ата

## Как работает

**Клиент:**
1. Выбирает район (Каракол, Чолпон-Ата...)
2. Выбирает категорию (Кафе, Мастер, Репетитор...)
3. Видит карточки → жмёт WhatsApp → пишет напрямую

**Бизнес/Мастер:**
1. Жмёт "Зарегистрировать бизнес"
2. Проходит 6 шагов (категория → район → имя → телефон → описание → адрес)
3. Ждёт одобрения (24 часа)
4. Получает уведомление и появляется в каталоге

**Блогер/Админ:**
- Получает заявки прямо в бот
- Одобряет / отклоняет одной кнопкой
- Команда /admin — вся статистика по районам

## Быстрый старт

### 1. Создай бота
[@BotFather](https://t.me/BotFather) → /newbot → сохрани токен

### 2. Узнай свой Telegram ID
[@userinfobot](https://t.me/userinfobot) → сохрани число

### 3. Настрой .env
```
cp .env.example .env
```
Заполни BOT_TOKEN, DATABASE_URL, ADMIN_IDS

### 4. Запусти
```bash
pip install -r requirements.txt
python main.py
```

## Деплой на Railway

1. Новый проект → GitHub репо
2. Добавь PostgreSQL
3. Переменные: BOT_TOKEN, DATABASE_URL (авто), ADMIN_IDS
4. Deploy ✅

## Команды администратора

| Команда | Что делает |
|---------|-----------|
| /admin | Статистика + разбивка по районам |
| /pending | Заявки на проверку (с кнопками) |
| /active | Все активные в каталоге |

## Структура файлов

```
kabarman-bot/
├── main.py
├── requirements.txt
├── railway.toml
├── .env.example
├── db/
│   ├── database.py      # Схема БД (с районами)
│   └── queries.py       # Все запросы
└── bot/
    ├── states.py        # FSM: поиск (район→категория), регистрация (6 шагов)
    ├── keyboards.py     # Все кнопки
    └── handlers/
        ├── start.py     # /start, о боте
        ├── search.py    # Поиск: район → категория → результаты
        ├── register.py  # Регистрация: 6 шагов с FSM
        └── admin.py     # Модерация + статистика
```
